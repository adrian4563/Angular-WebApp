# Ejercicio 5: Variables CSS + Angular Binding (Modo Oscuro/Claro)

## Descripción
Este ejercicio implementa un sistema de cambio de tema visual usando variables CSS y Angular binding.

## Funcionalidades
- Definición de variables CSS para colores principales
- Variables utilizadas en distintos componentes
- Botón para alternar entre modo claro y oscuro
- Cambio dinámico de estilos sin recargar página
- Uso de Angular para modificar clases/estilos globales mediante binding
- Interfaz que cambia de apariencia en tiempo real
- Persistencia del tema seleccionado (opcional)

## Archivos modificados
- src/styles.scss (variables CSS globales)
- src/app/app.component.ts
- src/app/app.component.html
- src/app/app.component.scss
- src/app/components/navigation/navigation.component.ts
