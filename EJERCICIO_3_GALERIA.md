# Ejercicio 3: Galería de Imágenes Interactiva

## Descripción
Este ejercicio implementa una galería de imágenes interactiva usando binding de propiedades en Angular.

## Funcionalidades
- Visualización de miniaturas de imágenes
- Imagen principal que se actualiza dinámicamente
- Clic en miniatura para cambiar imagen principal
- Imagen seleccionada resaltada visualmente
- Directivas y bindings:
  - [src] y [alt] para imágenes
  - [ngClass] para estilos dinámicos
  - *ngIf para mostrar/ocultar elementos
- Lógica dinámica de visualización
- Animaciones con Angular Animations
