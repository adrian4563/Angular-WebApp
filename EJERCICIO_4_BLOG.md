# Ejercicio 4: Sistema de Posts Dinámicos

## Descripción
Este ejercicio implementa un pequeño sistema de gestión de contenido (CMS) usando Angular con creación de estructuras dinámicas.

## Funcionalidades
- Formulario con campos: Título y Descripción
- Generación de "post card" al enviar formulario
- Posts agregados dinámicamente a la lista
- Componente independiente para cada post
- Visualización ordenada de múltiples elementos
- Mejoras adicionales:
  - Botones para editar publicaciones
  - Botones para eliminar publicaciones
  - Opción para destacar publicaciones
  - Modificación de contenido en tiempo real
